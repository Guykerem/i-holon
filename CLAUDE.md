# i-holon — Holonic Graph Architecture fork

This repository is a fork of Kurt Cagle's [holon-graph-shacl](https://github.com/kurtcagle/holon-graph-shacl), stress-tested as a candidate foundation for Ally's Kernel — the substrate underlying the Personal Context Graph.

**Upstream files at the repo root (`ontology/`, `shapes/`, `taxonomies/`, `README.md`, etc.) are pristine. Do not modify them.**

All of the Ally team's work lives in `ally/`. See `ally/CLAUDE.md` for the working charter.
