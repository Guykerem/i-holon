# ally/

The Ally team's stress-testing of the Holonic Graph Architecture (HGA) against the requirements of Personal Context Management.

Upstream (Kurt Cagle's `holon-graph-shacl`) is pristine. Our work lives here.

## Layout

- `CLAUDE.md` — working charter. Read this first.
- `models/` — code (TTL). Our evolution of upstream semantic artifacts.
- `ops/` — narrative (Markdown). Rationale, stress tests, mapping, examples, notes.

## The primary question

Does the 3×4 combination of **three spaces × four layers** give a **MECE decomposition** of **Integrated Individual Information (III)** — every piece has exactly one home, and nothing falls through the cracks?

See `ops/mapping/three-spaces-to-layers.md`.
